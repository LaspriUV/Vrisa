# VRISA — Vigilancia de Riesgos e Inmisiones de Sustancias Atmosféricas

Proyecto académico para el curso de Bases de Datos (Universidad del Valle).  
El objetivo general es diseñar y desarrollar una plataforma para el monitoreo de la calidad del aire en la ciudad de Cali, gestionando información proveniente de estaciones ambientales distribuidas.

## Descripción general

VRISA permite registrar instituciones, integrar estaciones de monitoreo y visualizar mediciones ambientales en dashboards, con el fin de analizar tendencias, generar reportes y apoyar la toma de decisiones.

Esta plataforma se centra en:

- Captura de mediciones de contaminantes criterio.
- Manejo de datos meteorológicos.
- Consulta histórica.
- Alertas y análisis comparativo.

## Objetivos principales

- Diseñar una base de datos relacional eficiente y normalizada.
- Implementar una API para la comunicación de las aplicaciones cliente.
- Desarrollar una interfaz que permita consultar información ambiental.
- Garantizar seguridad, trazabilidad y consistencia de los datos.

## Stack tecnológico (planeado)

- **Backend:** Python + Django (REST)
- **Base de datos:** PostgreSQL + PostGIS
- **Autenticación:** JWT
- **Frontend:** ReactJS + Bootstrap
- **Contenerización:** Docker & Docker Compose

(Tecnologías sujetas a pequeños ajustes durante el desarrollo)
